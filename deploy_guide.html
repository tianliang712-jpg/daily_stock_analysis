五步完成部署
1
上传代码到你的 GitHub 仓库
将本地文件推送到 GitHub（已准备好，直接用）

首先在 GitHub 上新建一个公开或私有仓库（建议私有，保护你的股票策略），然后把本地文件推送上去：

# 在 PowerShell 中执行（文件已在 WorkBuddy/daily_stock_analysis 目录） cd C:\Users\gftlwan\WorkBuddy\daily_stock_analysis # 初始化 Git（如果还没有） git init git add . git commit -m "feat: 初始化daily_stock_analysis项目" # 关联你的 GitHub 仓库（替换为你的仓库地址） git remote add origin https://github.com/你的用户名/daily_stock_analysis.git git branch -M main git push -u origin main
💡 或者直接在 GitHub 网页上新建仓库，然后把 src/analyze.py、requirements.txt、.github/workflows/daily_analysis.yml 三个文件手动上传也可以。
2
配置 Secrets 密钥（最重要！）
Settings → Secrets and variables → Actions → New repository secret
  1. 打开你的 GitHub 仓库页面
  2. 点击顶部 Settings（设置） 标签
  3. 左侧菜单找到 Secrets and variables → Actions
  4. 点击绿色按钮 "New repository secret"
  5. 依次添加下表中的 2 个密钥
密钥名称（Name）值（Secret）获取方式
GEMINI_API_KEY 形如 AIzaSy... 🔗 Google AI Studio 免费申请
（每分钟15次请求，完全够用）
WECHAT_WEBHOOK 形如 https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=... 企业微信群 → 右键群名 → 添加群机器人 → 复制 Webhook 地址
⚠️ 密钥名称必须完全一致（区分大小写）。不要在值中加引号。添加后只能看到名称、无法看到值，这是正常的安全设计。
✅ 股票池（66只A股）已内嵌在脚本中，无需配置 STOCK_LIST。
3
启用 GitHub Actions 工作流
Fork/新建仓库后 Actions 默认需要手动启用
  1. 点击仓库顶部 Actions 标签
  2. 如看到黄色提示框，点击 "I understand my workflows, go ahead and enable them"
  3. 左侧 Workflow 列表中点击 daily_stock_analysis
  4. 如看到 "This workflow is disabled" 提示，点击 Enable workflow
⏰ 工作流已设置为每个工作日 09:25（早盘前）和 15:30（收盘后）自动触发，无需其他操作。
4
手动触发测试（强烈建议）
确认系统正常工作，企业微信能收到消息
  1. 进入 Actions → daily_stock_analysis
  2. 点击右侧 "Run workflow" 下拉按钮
  3. 选择分支 main，点击绿色 "Run workflow" 确认
  4. 等待 2-5 分钟（AKShare 采集约60只股票需时），看到绿色 ✅ 说明成功
  5. 去企业微信群确认是否收到分析报告
🎉 若 Actions 显示绿色对勾，企业微信收到消息，部署完成！
🔍 若失败，点击红色 ❌ 进入日志，展开 "Run stock analysis" 步骤查看报错原因。

常见问题
❌ Actions 运行失败，红色叉叉怎么办？
点击红色 ❌ 进入日志 → 展开 "Run stock analysis" 步骤 → 查看最后几行报错。
常见原因：① GEMINI_API_KEY 拼写错误或已过期 → 重新添加 Secret； ② WECHAT_WEBHOOK 地址错误 → 检查是否完整复制； ③ AKShare 数据接口临时异常 → 等几分钟重新触发。
📵 没有收到企业微信消息？
① 确认 Webhook 地址完整（以 https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key= 开头）
② 在企业微信群内确认机器人没有被删除
③ Actions 日志里搜 "企业微信推送成功"，如果出现说明推送了但可能被过滤
➕ 想修改股票池（增删股票）怎么办？
编辑 src/analyze.py 中的 STOCK_POOL 字典，格式为 ("代码", "名称")。
A股代码规则：沪市主板6开头（如 600519）；深市主板0开头（如 002594）；创业板3开头（如 300750）。
注意：科创板688开头的股票数据接口可能有限制，建议不加。
⏰ 想修改触发时间怎么办？
编辑 .github/workflows/daily_analysis.yml 中的 cron 字段。
注意：GitHub cron 用的是 UTC 时间，北京时间（UTC+8）需减8小时。
例如北京时间 14:30 = UTC 06:30 = cron 写 30 6 * * 1-5
💰 GitHub Actions 会收费吗？
公开仓库：完全免费，无限使用。
私有仓库：每月免费 2000 分钟。本项目每次运行约 3-5 分钟，每月约 40 次，共约 200 分钟，远在免费额度内。
