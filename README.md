## 快速部署（5步搞定）

### 第一步：Fork 本仓库

点击右上角 **Fork**，将仓库复制到你的账号下。

### 第二步：配置 Secrets（密钥）

进入你 Fork 后的仓库：**Settings → Secrets and variables → Actions → New repository secret**

| 密钥名称 | 内容 | 获取方式 |
|---------|------|---------|
| `GEMINI_API_KEY` | 你的 Gemini API Key | [Google AI Studio](https://aistudio.google.com/app/apikey) 免费申请 |
| `WECHAT_WEBHOOK` | 企业微信机器人 Webhook | 企业微信群 → 添加群机器人 → 复制 Webhook 地址 |

> **注意**：`STOCK_LIST` 无需配置，股票池已内嵌在脚本中。

### 第三步：启用 Actions

1. 进入仓库 **Actions** 页
2. 点击 **"I understand my workflows, go ahead and enable them"**
3. 左侧找到 **daily_analysis.yml**，点击 **Enable workflow**

### 第四步：手动测试一次

**Actions → daily_analysis → Run workflow → Run workflow**

等待 2-3 分钟，去企业微信群查收报告！

### 第五步：自动定时运行

默认每个交易日触发两次：

| 时间（北京时间） | 用途 |
|---------------|------|
| **09:25** | 早盘开盘前分析，关注竞价方向 |
| **15:30** | 收盘后复盘，决策明日操作 |

## 常见问题

**Q：任务失败怎么办？**
- 检查 Secrets 是否正确（特别是 `GEMINI_API_KEY`）
- 进入 Actions 日志查看具体报错

**Q：没收到企业微信消息？**
- 确认 Webhook 地址完整，以 `https://qyapi.weixin.qq.com/...` 开头
- 在企业微信里确认机器人未被删除

**Q：想修改触发时间？**
- 编辑 `.github/workflows/daily_analysis.yml` 中的 `cron` 字段
- 注意 cron 是 UTC 时间，北京时间需 -8 小时

**Q：想增删股票怎么办？**
- 编辑 `src/analyze.py` 中的 `STOCK_POOL` 字典，格式为 `("代码", "名称")`
- A股代码：沪市6开头（如600519），深市0或3开头（如002594，300750）
